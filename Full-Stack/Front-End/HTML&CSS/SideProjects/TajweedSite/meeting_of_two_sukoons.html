The Meeting Of Two Sukoons In the Arabic language it is not possible for two saakin letters to be pronounced when found right next to each other. Note: One must pay close attention to special conditions that apply. To solve this problem of two saakin letters meeting, we will do either of the following depending on their conditions: 1) Remove the first saakin letter 2) Add a kasrah to the first saakin letter 3) Add a dhammah to the first saakin letter 4) Add a fathah to the first saakin letter 1) Removal of the first saakin letter: If the first saakin letter is a Madd letter (و ا ي), we will remove this first saakin letter in pronunciation, but not when writing it i.e: (قُلْنَا اهْبِطُوا, وَعَمِلُوا الْصَّالِحَاتِ , فِي الْأَرْضِ) It will instead be pronounced like the following: (قُلْنَ هْبِطُوا, وَعَمِلُ صَّالِحَاتِ , فِ لْأَرْضِ) 2) Add a kasrah to the first saakin letter: We add a vowel i.e.: fathah, kasrah or dhammah to the first saakin letter A) If the first saakin letter is a Non-Madd letter i.e.: (و ا ي), we will add a kasrah to this first saakin letter i.e: (قُلْ انْظُرُوا, أَنِ اعْبُدُوا) It will instead be pronounced like the following: (قُلِ نْظُرُوا, أَنِ عْبُدُوا) We will add a kasrah to the first saakin letter in both of the following examples since the first saakin letter is a Non-Madd letter و ا ي أَمْ ارْتَابُوْا , لِمَنْ ارْتَضَى They will be pronounced in the following way: أَمِ رْتَابُوْا , لِمَنِ رْتَضَى 2) Adding a kasrah to the first saakin letter (Tanween Example): We add a kasrah to the end of the tanween to prevent the two saakin letters from meeting. B) If the first saakin letter has a tanween i.e.: ( ٌ ٍ ً ), we will drop one of the vowels of the tanween i.e.: (fathatain will only be fatha/kasratain will only be kasrah/dhammatain will only be dhammah) and then add a noon to it with a kasrah i.e: (إِلَى رَبِّكَ يَوْمَئِذٍ المُسْتَقَرُّ, خَيْرًا الوَصِيَّةُ) It will instead be pronounced like the following: (إِلَى رَبِّكَ يَوْمَئِذِنِ لْمُسْتَقَرُّ, خَيْرَنِ لْوَصِيَّةُ) The Meeting Of Two Sukoons - More Tanween Examples: We will drop one of the vowels from the tanween and add a kasrah to the end of the tanween to prevent the two saakin letters from meeting. (لَهْوً انْفَضُوا, أَحَدٌ اللهُ الْصَّمَدُ) It will instead be pronounced like the following: (لَهْوَنِ نْفَضُوا, أَحَدُنِ للهُ صَّمَدُ) 3) Adding a dhammah to the first saakin letter: If the first saakin letter is a plural meem or waaw used as a pronoun, we will remove the sukoon from it and add a dhammah i.e: (اُدْخُلُوا عَلَيْهِمْ الْبَاب, فَتَمَنَّوْ الْمَوْتَ) It will instead be pronounced like the following: (اُدْخُلُوا عَلَيْهِمُ لْبَاب, فَتَمَنَّوُ لْمَوْتَ) 4) We will add a fathah to the first saakin letter in the following cases: A) If the first word that comes is مِنْ/إِلَىi.e: (مِنْ اللهِ , مِنْ الْشَّاهِدِيْنَ , إِلَى اللهِ) It will instead be pronounced like the following: (مِنَ للهِ, مِنَ شَّاهِدِيْنَ, , إِلَ للهِ) B) When الله comes right after الم like in Surat Aali ‘Imraan, we will add a fathah to the meem when connecting the two verses together i.e: (الم اللهُ لَا إِلَهَ إِلَّا هُوَ الْحَيُّ الْقَيُّوْمُ) C) The taa will always have a fatha before an Alif i.e: (كَانَتَا تَحْتَ عَبْدَيْنِ مِنْ عِبَادِنَا) It will instead be pronounced like the following: (كَانَتَا تَحْتَ عَبْدَيْنِ مِنْ عِبَادِنَا)
