The Two Types Of Hamza (ء ا) Hamzatul-Wasl: (ا) this is the type of hamza which will be skipped if a letter came before it; whether in the same word or another word, however this hamza will be pronounced if no letter came before it i.e: (اِسْمٌ, عِيْسَى ابْنُ مَرْيَمَ, عُمَرُ ابْنُ الخَطَّاب, مَا اسْمُكَ) Hamzatul Qat’:(ء) this is the type of hamza that will always be recited clearly; whether in the beginning of a word or in the middle of a word i.e: (أَبَدًا, أَحْمَدُ, يَأْكُلُ )
