Meem Saakinah Rules Meem Saakinah is a meem that is vowelless which appears in the middle of a word. The sign of “sukoon” is ( ْ ) Note: All Meem Saakinah rules apply to meem with sukoon. There are 3 main rules for Meem Saakinah Al-Idghaam Mithlayn Sagheer الإِدْغَامُ (Small merging of two of the same letters) Al-Ikhfa Ash-Shafawi الإِخْفاَءُ الشَّفَوِي(Labial Concealment) Al-Ith’har Ash-Shafawi الإِظْهَارُ الشَّفَوِي (The Clear)
